Detailed Description

Represents non-constructable registry.

Author
Rafael Plugge

Contains a single static instance of our registry to make access to the registry easier as well as utilise the singleton pattern to garantee only one registry is ever created.

A registry will store all our entities and their components in highly optimised sets to allow for fast retrieval and iteration of several entities and different combinations of componenents.
