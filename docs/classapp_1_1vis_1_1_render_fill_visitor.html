Detailed Description

Used when needing to access our render fill variant.

Author
Rafael Plugge

When utilizing the standard libraries variant to maintain type safety a visitor class is needed, this being one, is utilised to access the variant and assign appropriatly to app::vis::RenderFillVisitor::m_rectangleShape.
